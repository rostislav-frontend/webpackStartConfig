# WEBPACK



## Установка пакетов

### npm
```shell
npm install
```
### yarn
```shell
yarn
```
### pnpm
```shell
pnpm
```

## Команды

### Запуск сервера для разработки
```shell
npm run start
```

### Сборка проекта без оптимизации
```shell
npm run build:dev
```

### Сборка проекта с оптимизацией
```shell
npm run build:prod
```

### Очистка папки dist
```shell
npm run clear
```
